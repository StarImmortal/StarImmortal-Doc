# 常用正则表达式

1. 验证除了0之外的任何整数

```js
/^(\+|-)?[1-9][0-9]*$/
```

2. 验证除了0之外的任何数字

```
/^(\+|-)?([1-9][0-9]*(\.\d+)?|(0\.(?!0+$)\d+))$/
```

3. 只能数字（允许两位小数）

```js
/(^[1-9]([0-9]+)?(\.[0-9]{1,2})?$)|(^(0){1}$)|(^[0-9]\.[0-9]([0-9])?$)/
```

4. 非负整数

```js
/^\d+$/
```

5. 大于0的正整数

```js
/^\+?[1-9]\d*$/
```

6. 匹配Email地址

```js
/\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*/
```

7. 匹配手机号码

```js
/^(13[0-9]|14[01456879]|15[0-35-9]|16[2567]|17[0-8]|18[0-9]|19[0-35-9])\d{8}$/
```

8. 匹配身份证号码

```js
/^(^[1-9]\d{7}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}$)|(^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])((\d{4})|\d{3}[Xx])$)$/
```
